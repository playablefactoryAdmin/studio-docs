# Asset Viewer

import Hoverbox from "@components/Hoverbox";
import Image from "next/image";

**Asset Viewer** is the main dockable panel that lets you navigate to various other panels, 
including the **Assets Panel** shown in the example below. It's likely the panel you'll use most frequently.

Click on the bubbles to go the related documentation page.
