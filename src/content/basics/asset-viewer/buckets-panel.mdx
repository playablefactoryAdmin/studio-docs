# Buckets Panel

Before continuing, we recommend reading the [Buckets](/studio-docs/advanced/buckets.mdx) page for important background information.

To create a Bucket, click the "+" button in the Buckets tab of the Assets Viewer window and then select the object type you want to create.

<Image src="/studio-docs/advanced/buckets/create.gif" alt="create" width={932} height={435} className="mt-6 rounded-xl overflow-hidden" />

By clicking on the Bucket you created, you can configure its properties, which appear in the Inspector window. For more information on the properties, see the [Buckets](/studio-docs/advanced/buckets.mdx) page.
