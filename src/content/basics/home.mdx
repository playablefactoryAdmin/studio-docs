## Projects

Clicking on the **Projects** tab displays your most recently opened projects. You can use the search bar at the top to quickly find a specific project.

To the right of the search bar, you’ll find buttons to **import** an existing project or **create a new** one.

## Store

The **Store** tab provides the same functionality as accessing the Store within a project.
For detailed information, see the [Toolbox → Store](toolbar.mdx#store) section.

## Learn

The **Learn** tab opens this documentation site directly within the Studio application.
