# AI Types

In the Data.js library, you'll find a set of data types designed for AI-generated content — voice, audio layers and avatars. This section provides documentation for the following AI types:

- [Voiceover](ai-types/voiceoverdata.md)
- [AudioWithAI](ai-types/audiowithaidata.md)
- [AIAvatar](ai-types/aiavatardata.md)

## Voiceover

The `Voiceover` data type is used to configure AI-generated or uploaded voiceover tracks (script, speaker, volume, source).

[Learn more about Voiceover Data](ai-types/voiceoverdata.md)

## AudioWithAI

The `AudioWithAI` data type combines multiple audio channels (background song, AI sound design, custom track, SFX) into a single parameter.

[Learn more about AudioWithAI Data](ai-types/audiowithaidata.md)

## AIAvatar

The `AIAvatar` data type configures an AI-driven on-screen avatar (actor, position, source asset).

[Learn more about AIAvatar Data](ai-types/aiavatardata.md)
