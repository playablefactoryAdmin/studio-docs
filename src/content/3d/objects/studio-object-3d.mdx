`StudioObject3D` is a **type**, not a class. It is defined as:

```ts
export type StudioObject3D = ThreeObject3D & THREE.Object3D;
```

This means it extends the native `THREE.Object3D` with additional properties and methods defined in the `ThreeObject3D` type.

### ThreeObject3D

```ts
type ThreeObject3D = {
	sID: string;
	type: string;
	components: Components3D;
	updateComponents: (components: any) => void;
	getComponent: (component: string) => any;
	selected: boolean;
	locked: boolean;
};
```
