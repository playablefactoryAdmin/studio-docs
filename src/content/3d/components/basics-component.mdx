### Label

Sets the display name or label of the object.

### Visible

Toggles the visibility of the object on or off.

### Position

Set the position of the object.

### Scale

Set the scale of the object.

### Rotation

Set the rotation of the object (in degrees).

### Render Order

This value allows you to override the default rendering order of objects in the scene graph, while still maintaining separate sorting for opaque and transparent objects.

For more information, check [Three.js documentation](https://threejs.org/docs/#api/en/core/Object3D.renderOrder)
