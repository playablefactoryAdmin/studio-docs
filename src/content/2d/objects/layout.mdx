Create a `Layout2D` object by right clicking in the hierarchy and selecting `Add Node` → `Core` → `Layout2D`.

The `Layout2D` object manages a flexible 2D layout system with support for portrait and landscape modes, child resizing, and debug features.

It automatically resizes child objects and updates layout based on parent size changes.
