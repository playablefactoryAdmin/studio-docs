Create a `Text` object in **Studio** by right-clicking in the hierarchy and selecting `Add Node` → `Core` → `Text`.

This object allows you to render styled text using `PIXI.Text`, with full support for dynamic updates, shadows, strokes, word wrapping, auto-fitting text to its container, and responsive behavior.

Text content and style can be provided either as plain string or as a structured object with additional styling fields.

```ts
text.setText({
  text: "Welcome!",
  color: "#ffcc00",
  align: "center",
  multiline: { enabled: true, lineHeight: 1.5 },
  stroke: { color: "#000", thickness: 3 },
  shadow: { color: "#000", alpha: 0.5, angle: 45, distance: 4, blur: 2 }
});
```

---

#### setText

```ts
setText(text: string | TextData): void
```

Sets text content. If a `TextData` object is passed, it applies styles as well.

---

#### setFont

```ts
setFont(font: string): void
```

Sets the font family, validating it against loaded custom fonts.

---

#### updatePixiObject

```ts
updatePixiObject(): void
```

Applies internal style properties to `PIXI.TextStyle`, triggers layout recalculation, and handles autofit if enabled.

---

#### fitText

```ts
fitText(): void
```

Dynamically adjusts font size to fit within parent bounds based on `autofit` ratios.

---

#### updateTextComponent

```ts
updateTextComponent(component: TextGC): void
```

Applies values from a TextGC component. Also resolves template-linked data if applicable.

---

#### getTextComponent

```ts
getTextComponent(): TextGC
```

Returns a snapshot of the current text style and config as a `TextGC`.

---

#### setTextStyles

```ts
setTextStyles(textData: TextData): void
```

Applies individual style values from a `TextData` object (e.g., fill, stroke, shadow).
