Create a `Graphics` object by right clicking in the hierarchy and selecting `Add Node` → `Core` → `Graphics`.

The Graphics object allows drawing of various shapes like rectangles, circles, ellipses, stars, lines, and polygons. It supports fill and stroke styling, and integrates with Studio’s component system to update its content dynamically.
