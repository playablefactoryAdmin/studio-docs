Create a `TilingSprite` object by right-clicking in the hierarchy and selecting `Add Node` → `Core` → `TilingSprite`.

The TilingSprite object displays a texture repeated (tiled) across its dimensions. It supports texture updates, tile positioning, scaling, tinting, and landscape texture support.

It wraps PIXI's `TilingSprite` and integrates with Studio’s component system.
