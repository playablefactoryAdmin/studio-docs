Create a `NineSliceSprite` object by right-clicking in the hierarchy and selecting `Add Node` → `Core` → `NineSliceSprite`.

This object displays a nine-slice scaled sprite using a texture, allowing scalable UI elements without distortion of corners.

It supports texture swapping, tinting, custom slice sizes, and landscape texture support.
