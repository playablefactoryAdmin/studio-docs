Create a `Video` object by right-clicking in the hierarchy and selecting `Add Node` → `Core` → `Video`.

The Video object wraps an HTMLVideoElement as a PIXI.Texture, allowing video playback inside the Studio engine.

It supports playback controls, looping, muting, autoplay, and unique texture keys for multiple instances.
