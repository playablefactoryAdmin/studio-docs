Create an `HTMLText` object by right clicking in the hierarchy and selecting `Add Node` → `Core` → `HTMLText`.

The HTMLText object renders styled HTML content within the Studio engine using PIXI’s `HTMLText` class. It supports font styling, alignment, stroke, shadow, word wrap, and integrates with Studio’s component system for dynamic text updates.
