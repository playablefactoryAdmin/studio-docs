Create a `Container` in **Studio** by right clicking on the hierarchy and selecting `Add Node` -> `Core` -> `Container`.

Studio automatically generates a container instance for you, based on the `PIXI.Container` class. 
This instance includes all the core functionality of `PIXI.Container`, along with additional extensions designed to enhance usability.

#### updateResponsive2D
```ts
updateResponsive2D(index?: number, direction?: string, data?: any, animated?: boolean)
```

Changes and updates the responsive2D component data.

<InnerHeader title="Parameters"/>

<UserTable data={
    [
        {name: "index", desc: "Responsive2D index."},
        {name: "direction", desc: "Portait or landscape. Defaults to portrait."},
        {name: "data", desc: "Responsive2D data. An example with all the possible values is shown in the example usage section."},
        {name: "animated", desc: "Determines whether the responsive2D is updated with animation. Defaults to false."},
    ]
} />

<InnerHeader title="Example Usage"/>

```js
this.node.updateResponsive2D(
    0, 
    "portrait", 
    { 
		fitType: "min"; //min, max, stretch
		width: 25;
		height: 25;
		origin: "top"; //top-left, top, top-right, left, center, right, bottom-left, bottom, bottom-right
		horizontalSpace: 10;
		horizontalSpaceUnit: "self"; // %, px, self
		verticalSpace: 10;
		verticalSpaceUnit: "self";
		enabled: true;
    }, 
    false
);
```
