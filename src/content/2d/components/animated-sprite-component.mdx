# Animated Sprite Component

import Image from "next/image";

This component is automatically added to all Animated Sprite objects.

<Image src="/studio-docs/components/animated-sprite-component.png" alt="Animated Sprite Component" width={600} height={600} className="pt-6" />

### Atlas Texture  
The sprite sheet used for rendering animation frames.

### Animation  
The name of the animation to play from the atlas.

### Auto Play  
Whether the animation should automatically start playing when the object is added to the scene.

### Loop  
Determines if the animation should loop continuously.

### Animation Speed  
Controls the playback speed of the animation.

### Tint  
Applies a color tint to the sprite.
