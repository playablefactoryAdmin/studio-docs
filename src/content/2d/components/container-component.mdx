### Size

Automatically calculates the width and height of the container given its children.

### Debug

Enables the debug container that shows the bounding box of the container.

### Width

Width of the container.

### Height

Height of the container.

### Landscape

Enabling it lets the user set custom width and height parameters for the container on landscape mode.
