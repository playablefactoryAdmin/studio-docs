### Texture 

You can set the texture of the sprite by changing the texture property. It is also possible to drag and drop the texture from the Asset Viewer.

### Bucket

Buckets are utilized to load a sprite based on a specified parameter. For more information about buckets, please check out the [Buckets](/basics/buckets) documentation.

<Image src="/studio-docs/components/sprite-bucket.png" alt="Sprite Bucket" width={400} height={400} className='pt-6 '/>

After creating a bucket, select the bucket inside **Studio**, and specify the parameter name in the Data.js ID field.

### Uploadable

Uploadable tab is used to make a sprite uploadable on the platform. After enabling the uploadable option, you will need to specify 3 parameters:

<Image src="/studio-docs/components/sprite-uploadable.png" alt="Sprite Uploadable" width={400} height={400} className='pt-6 '/>

#### Default Texture
This texture will be displayed when the user has not uploaded a custom one on the platform.

#### DataJS Parameter ID
A unique identifier that developers must assign to the parameter. This value is not visible to end users on the platform.

#### DataJS Name
The display name of the parameter as it will appear to users on the platform.
