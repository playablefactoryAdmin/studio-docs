# Spine Component

import Image from "next/image";

This component is automatically added to all Spine objects.

<Image src="/studio-docs/components/spine-component.png" alt="Spine Component" width={600} height={600} className="pt-6" />

### Spine Texture

The texture atlas and skeleton data used to render the Spine animation.

### Skin

The skin to apply to the skeleton. Allows for visual variations without modifying the rig.

### Animation

The name of the animation to play on the skeleton.

### Auto Play

Whether the animation should start automatically when the object is added to the scene.

### Loop

Determines if the animation should repeat continuously.

### Animation Speed

Controls how fast the animation plays.

### Tint

Applies a color tint to the entire Spine object.
