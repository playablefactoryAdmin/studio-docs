# Filters Component

To add this component, click on `Add Component` on Inspector, then select `Filters2D`.

<Image src="/studio-docs/components/filters-component.gif" alt="filters component" width={500} height={300} className="mt-6 rounded-xl overflow-hidden" />

Uses `pixi-filters` module. Adds the selected filter to the selected game object. You can use multiple filters on a single filter component.
